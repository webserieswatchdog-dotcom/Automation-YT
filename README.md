## Installation

```bash
git clone https://github.com/webserieswatchdog-dotcom/Automation-YT.git
cd Automation-YT
pip install -r requirements.txt
```

## Usage

```bash
python src/main.py
```

## Contributing

Contributions are welcome. Please read `CONTRIBUTING.md`.



## Vision

Automation-YT aims to provide an open-source framework for AI-powered YouTube content automation.

The goal is to help creators automate repetitive tasks using Python and open-source AI technologies.
